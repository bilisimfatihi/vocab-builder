# VocabBuilder 📚

VocabBuilder is a modern web application designed to help users digitize their language learning process, create personal dictionaries, and improve their vocabulary through interactive training modules.

## 🚀 Project Overview

This project was developed as a capstone/portfolio work within the GoIT educational program. It features a comprehensive system for user authentication, dynamic data listing, filtering, and a specialized learning dashboard.

## 🛠 Tech Stack

* **Core:** React (with TypeScript)
* **Build Tool:** Vite
* **State Management:** Redux Toolkit
* **Form Management:** React Hook Form & Yup (for Validation)
* **API Handling:** Axios
* **UI Components:** Material UI (MUI) & Custom Styles
* **Deployment:** Netlify / GitHub Pages

## 🌟 Key Features

* **Authentication:** Secure Register and Login systems with specific validation patterns.
* **Interactive Dashboard:** Real-time statistics and advanced filtering with Debounce technique.
* **Dictionary Management:** Comprehensive CRUD operations (Add, Edit, Delete) for words.
* **Training Module:** Progress-tracked interactive tests to reinforce learning.
* **Responsive Design:** Fully adaptive for Mobile (320px+), Tablet (768px+), and Desktop [1440px+]

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone <repo-url>
  ```

1. Install dependencies:

```bash
npm install
```

1. Run the application in development mode:

```bash
npm run dev
```
